═══════════════════════════════════════════════════════════════
MESSAGE FLOW EXAMPLES
═══════════════════════════════════════════════════════════════

1. LIST APPLICATIONS
   ─────────────────
   Client                          Server
     │──REQUEST: LIST_APPS────────→│
     │                              │ [Execute PowerShell]
     │                              │ Get-Process | Where MainWindowTitle
     │←─RESPONSE: LIST_APPS─────────│
     │  {apps: [...]}               │


2. SCREENSHOT
   ──────────
   Client                          Server
     │──REQUEST: SCREENSHOT────────→│
     │                              │ [Robot.createScreenCapture()]
     │                              │ [Encode to JPEG]
     │←─RESPONSE: SCREENSHOT────────│
     │  {image: base64, size: ...}  │


3. FILE TRANSFER (CHUNKED)
   ────────────────────────
   Client                          Server
     │──REQUEST: DOWNLOAD_FILE─────→│
     │  {fileId, path}              │
     │                              │ [Open file]
     │←─STREAM: FILE_INFO───────────│
     │  {fileSize, totalChunks}     │
     │                              │ [Read chunk 0]
     │←─STREAM: FILE_CHUNK_0────────│ (1MB)
     │←─STREAM: FILE_CHUNK_1────────│ (1MB)
     │      ... 1024 chunks ...     │
     │←─STREAM: FILE_CHUNK_1023─────│ (1MB)
     │  [FileChunkReceiver.addChunk()]
     │  [Reassemble when complete]  │
     │←─RESPONSE: TRANSFER_COMPLETE─│


4. WEBCAM STREAMING
   ─────────────────
   Client                          Server
     │──REQUEST: START_WEBCAM──────→│
     │                              │ [Start webcam thread]
     │←─STREAM: WEBCAM_FRAME────────│ (JPEG 30KB)
     │←─STREAM: WEBCAM_FRAME────────│ (15 FPS)
     │←─STREAM: WEBCAM_FRAME────────│
     │      ... continuous ...      │
     │──REQUEST: STOP_WEBCAM───────→│
     │                              │ [Stop thread]
     │←─RESPONSE: STOP_WEBCAM───────│


5. CONCURRENT OPERATIONS
   ──────────────────────
   Client                          Server
     │──REQUEST: START_WEBCAM──────→│
     │                              │ Thread 2: [Stream webcam]
     │←─STREAM: WEBCAM_FRAME────────│  ↓
     │                              │  ↓
     │──REQUEST: LOCK_SYSTEM───────→│ Thread 1: [Lock keyboard/mouse]
     │←─RESPONSE: LOCK_SYSTEM───────│  ↑ (immediate)
     │                              │  ↓ (still streaming)
     │←─STREAM: WEBCAM_FRAME────────│  ↓
     │                              │  ↓
     │──REQUEST: KILL_PROCESS──────→│ Thread 1: [Kill process]
     │←─RESPONSE: KILL_PROCESS──────│  ↑ (immediate)
     │←─STREAM: WEBCAM_FRAME────────│  ↓ (still streaming)
